Getting started

Importing lootalot

Using NodeJS

lootalot is available on the npm registry—you can use it to install lootalot like how you would install other npm packages: 

npm install lootalot

From there, it's just a matter of importing lootalot's classes using whatever method you want to use: 

// CommonJS
var { LootTable } = require("lootalot");
// ES6 import
import { LootTable } from "lootalot";

On the browser

You can use services like unpkg and jsdelivr to use lootalot on the web: 

<!-- unpkg -->
<script src="https://unpkg.com/lootalot@1/dist/lootalot.min.js" ></script>
<!-- jsdelivr -->
<script src="https://cdn.jsdelivr.net/npm/lootalot@1/dist/lootalot.min.js" ></script>

All of lootalot's classes will be inside the lootalot JS object: 

// Retrieve LootTable class
const { LootTable } = lootalot;

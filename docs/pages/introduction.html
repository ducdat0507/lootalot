Introduction

lootalot is a JavaScript library that specializes in simulating loot drops from loot tables. Additionally, it uses probability math to approximate arbitrarily high numbers of trials which can be helpful if your use case involves a ton of looting simutaneously.

Example usage

// Import our LootTable class.
import { LootTable } from 'lootalot';

// Make a loot table for our example monster.
let table = new LootTable(
    // Each one of these pairs of square brackets 
    // represents a separate loot pool a roll
    // will draw independently.
    [
        // Drop one of these items per roll.
        { item: "Apple" },
        { item: "Banana" },
        { item: "Citrus" },
        // These items are 10 times rarer than
        // those above.
        { item: "Diamond", w: 0.1 },
        { item: "Emerald", w: 0.1 },
    ],
    [
        // Additionally, drop an extra 150~250 gold.
        { item: "Gold", count: [150, 250] }
    ],
    [
        // And an additional 1% chance for our monster
        // to drop 1~5 gem.
        { item: "Gem", count: [1, 5], p: 0.01 }
    ]
);

// Kill our monster one trillion times, why not?
let loot = table.loot(1_000_000_000_000);

// Display our loot.
console.table(loot);

What is this for?

This library is made for games in mind, specifically incremental/idle games that might need to simulate a ton of loot simutaneously as part of their offline progression calculation.
